 
Pastebin-Lite is a lightweight RESTful web application that allows users to create, share, and retrieve text pastes via unique URLs. Each paste can automatically expire based on time-to-live (TTL) or maximum view count, ensuring secure and temporary data sharing.

This project demonstrates real-world backend development, REST API design, database persistence, and deployment readiness using Java & Spring Boot.
